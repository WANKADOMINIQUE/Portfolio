# Portfolio
A full-stack professional portfolio web application built with Vue.js, PHP (Laravel), Tailwind CSS, and MySQL.
Designed to showcase software engineering skills, projects, work experience, and technical expertise in a modern and scalable way.
Features a dynamic admin dashboard for managing projects, blogs, testimonials, skills, and contact messages.
Includes responsive UI/UX, authentication, RESTful APIs, and database-driven content management.
Built with clean architecture, performance optimization, and production-ready best practices.
